About Us

Women Safety Using Machine Learning is an intelligent system designed to improve women's security through advanced technology. The platform analyzes historical crime data using machine learning algorithms to identify high-risk locations and predict possible crime patterns.

Our mission is to empower women by providing a smart safety framework that offers awareness about dangerous areas and allows quick emergency responses. The system helps users stay informed and protected using data-driven insights.

This project demonstrates how Artificial Intelligence and Machine Learning can be applied to solve real-world social problems and contribute to building a safer society for women.
